rust-aobench
============

![aobench](https://raw.githubusercontent.com/sharow/rust-aobench/master/target/imgs/image.png)

[aobench](http://code.google.com/p/aobench/) @ Rust


## Build & Run
```
$ rustc -v
rustc 0.13.0-nightly

$ rustc --opt-level 3 -o target/aobench src/main.rs
$ ./target/aobench

### if you have Cargo
$ cargo run --release
```
