rust-aobench
============

[aobench](http://code.google.com/p/aobench/) @ Rust


## Build
```
$ rustc -v
rustc 0.12.0-dev

$ rustc --opt-level 3 -o target/aobench src/main.rs
$
$ # if you have Cargo
$ cargo build

```

## Run
```
$ ./target/aobench
```

[aobench]: http://code.google.com/p/aobench/ "aobench"
